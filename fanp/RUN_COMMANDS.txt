WHAT GETS SAVED NOW

Main experiment saves:
- results\main_<timestamp>.json
- results\main_<timestamp>_summary.txt
- meta.run_id + meta.repro metadata
- meta.dense_test_acc
- meta.config
- param_total, param_nonzero, compression_ratio for each method
- wall_clock_sec for each method
- step_acc recovery traces
- FANP round_log with mean_fs, rate_used, n_pruned
- pruned FANP models in checkpoints\pruned\fanp_sp*_<run_id>.pth

Ablation saves:
- results\ablation_<timestamp>.json
- top-level meta (run_id, repro, config)
- round_log per ablation config
- step_acc recovery traces
- param_total and param_nonzero
